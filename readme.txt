Чтобы открыть данный сайт, необходимо:
1.Скачать Denwer. Ссылка -http://www.denwer.ru/. 
После выбора версии заполняем все необходимые поля на открывшейся странице и ждем ссылки на скачку выбранного дистрибутива программы. После этого скачиваем файл по ссылке и запускаем установку.
Следуем написанному в окне команд.
2. Протестировать Denwer. 
Сразу после запуска в правом нижнем углу монитора рядом с часами появятся два значка программы. Если они появились – значит программа запущена.
Открываем браузер и переходим по адресу http://localhost.
Ниже на странице можно будет найти блок ссылок для проверки корректности работы phpMyAdmin, доменов, SSL и т.д. Для проверки достаточно перейти по ссылке из списка. Если страница загрузилась корректно – значит все работает как следует.
3. Скопировать файлы сайта с диска и поместить их по такому пути: Z\WebServers\home\test1.ru\www\simple-website
4. В браузере перейти по адресу http://test2.ru/simple-website/.
